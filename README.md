# TSARMIM
The public data of "Rejection and Mitigation of GNSS Time Synchronization Attacks Based on Iterative Model"  <br>

The dataset has made 9 cases data publicly available. One of them is the TEXBAT ds2 data (https://radionavlab.ae.utexas.edu/texbat/).  At the same time, we would like to **express our gratitude to the Texbat dataset** for its contribution to the development of GNSS anti-spoofing technology.  <br>

Here, only the data used by the TSARMIM is presented to demonstrate the reproducibility of the paper. The algorithm code is also made public. **The relevant code has been patented, so please refrain from using it commercially.**  <br>

The parameters of the 8 experimental groups are detailed in the paper.  <br>

The required code for TSARMIM can be found in TEXBAT_ds2.rar.  <br>

We modify the SDR (https://github.com/gnsscusdr/CU-SDR-Collection) for TSARMIM. See 'modify for SDR'.
